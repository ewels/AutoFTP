Create New Site

Fill in the details below to create a new FTP site. Login details for the newly created site will be shown on screen and sent to you in an email which you can forward to your collaborators.

Please only request a site for as long as you actually need it since space on the server is limited and data which is no longer required needs to be cleaned up. You will be notified just before your site is removed - and you will have the option to extend the site at that time.

If you need a site to be available for a much longer period then please contact bioinformatics who can arrange for custom accounts to be set up on this system.
